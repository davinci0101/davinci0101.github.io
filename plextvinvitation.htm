Plex.tv 가입 및 공유 초대 수락 방법

👇 아래로 스크롤하여 내용을 확인하세요

  • Plex.tv 가입방법

    Plex.tv에 무료로 가입하여 공유된 미디어를 사용할 수 있습니다.

    현재 국내법(개인정보보호법)에 따라 국내에서 가입이 제한되어 있으므로 VPN프로그램을 통해 다른 나라에서 가입하는 형태로 제한적인 가입이 가능합니다.

    본 설명서는 무료vpn을 이용하여 plex.tv에 가입하는 방법을 다루고 있습니다.

  • 1.검색포탈에서 검색

    검색포탈에서 "browsec vpn"을 검색합니다.

  • 2.검색결과 확인

    검색결과 중 크롬 스토어 연결페이지를 선택합니다.

  • 3.확장 프로그램 설치

    추가 버튼을 클릭하여 확장프로그램을 설치합니다.

  • 4.VPN 실행

    설치된 확장프로그램에서 "Start VPN" 버튼을 클릭합니다.

    프로그램이 VPN 연결을 시도합니다.

  • 5.VPN 실행 확인

    VPN이 정상적으로 동작하면 연결된 나라가 표시되고 우측 하단 버튼 부분이 ON으로 표시됩니다.

  • 6.Plex.tv 가입

    www.plex.tv/sign-up 으로 이동합니다.

    email 주소 및 패스워드를 입력하고 "sign up with email" 을 클릭합니다.

  • 7.Plex.tv 계정 생성

    "Create an Account" 버튼을 클릭하여 계정을 생성합니다.

  • Plex.tv 계정 확인

    계정이 생성되면 자동으로 로그인 되면서 우측 상단에 내용이 표시됩니다.

    ⚠️ 생성된 계정은 이메일을 통해 확인절차를 완료하여야 합니다.

  • 이메일 확인

    Plex.tv 에서 보낸 이메일이 등록된 이메일에 도착해 있을 것입니다.

    내용 중 "Verify Email" 버튼을 클릭하여 확인합니다.

  • 공유 수락

    미디어 공유자에게 공유 요청을 하면 공유자는 최초 1회 초대 메일을 발송합니다.

    메일 본문에서 "ACCEPT"를 클릭하여 공유를 수락합니다.

    이후의 공유는 공유 수락 메일을 신경 쓰지 않으셔도 됩니다.
